# Ransam Healthcare Portfolio

A modern, interactive portfolio website for Ransam Healthcare products with smooth page-flip animations.

## Project Structure

```
DEMO1/
├── index.html       # Main HTML file
├── styles.css       # All CSS styles and animations
├── script.js        # JavaScript for navigation and interactions
└── README.md        # This file
```

## Features

- **Responsive Design**: Clean, modern layout optimized for viewing
- **Page Flip Animation**: Smooth 3D flip transitions between product pages
- **Navigation Sidebar**: Easy access to all product pages
- **Product Showcase**: Individual pages for each healthcare product
- **Dark Theme**: Professional dark color scheme

## How to Use

1. Open `index.html` in any modern web browser
2. Click on the navigation buttons on the right sidebar to view different products
3. Enjoy the smooth page-flip animations between sections
