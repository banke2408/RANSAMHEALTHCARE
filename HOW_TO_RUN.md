### Method 3: Using Python (For Developers)

If you have Python installed:

**On Mac/Linux:**
```bash
cd /path/to/DEMO1
python3 -m http.server 8000
```

**On Windows:**
```cmd
cd C:\path\to\DEMO1
python -m http.server 8000
```

Then open browser and go to: **http://localhost:8000**

---

### Method 4: Using Node.js (For Developers)

If you have Node.js installed:

```bash
# Install http-server (one-time only)
npm install -g http-server

# Navigate to project
cd /path/to/DEMO1

# Start server
http-server -p 8000
```

Then open browser and go to: **http://localhost:8000**

---

### Method 5: Using VS Code (For Developers)

1. Install **Live Server** extension in VS Code
2. Right-click `index.html`
3. Select **"Open with Live Server"**
4. Website opens automatically!
